# SDP 2026 - Generative Agentic AI Projects

This repository contains my mini projects developed during the Skill Development Program (SDP) 2026, focusing on **Generative Agentic AI Using Python**.

## Projects Overview

### 1. AGENTIC_AI
Foundational projects exploring AI systems, chatbots, and agentic workflows using Python.

### 2. CodeReviewMentor
An AI-powered code review assistant that provides feedback based on industry standards (Google API Design, Python Style Guide, Security Standards).

### 3. elearning-chatbot
Interactive chatbot implementations for e-learning scenarios, including RAG-based systems and memory-enabled conversational agents.

### 4. HireReady
Interview preparation assistant leveraging company-specific guidelines (Amazon Leadership Principles, Google Interview Standards, etc.).

## Technologies Used
- Python
- LangChain
- Google Generative AI (Gemini)
- ChromaDB (Vector Database)
- Streamlit
- Ollama

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` files with your API keys (see `.env.example` if provided)
4. Run individual projects as needed
